# `ux/first-time-user-walkthrough-feed-connect-convey`
```
DNA includes a first-time user walkthrough experience for new users landing on the feed after completing onboarding and profile setup. The walkthrough provides step-by-step education on the three main beta features: DNA Feed (discovering and engaging with diaspora content), DNA Connect (discovering and connecting with diaspora members), and DNA Convey (creating and sharing impact stories). Users can interact with and "touch and feel" key components: how to message people, view stories, and post content. The walkthrough displays exactly once per user upon first feed access, then never appears again. This progressive disclosure approach educates users on core features without blocking their ability to explore the platform.
```

# `ux/coming-soon-landing-pages-convene-collaborate-contribute`
```
DNA includes beautiful, detailed "Coming Soon" landing pages for three deferred features (Convene, Collaborate, Contribute) unavailable during beta period. Rather than showing empty pages or construction placeholders, these pages display compelling explanations of each feature's purpose and how they connect to the overall diaspora mobilization mission. Users encounter these landing pages when clicking navigation buttons for deferred features, managing expectations about beta scope while maintaining engagement with the platform's long-term vision. User explicitly requested these pages explain "what they will do for them over the next additions that we will make on the platform" to prevent negative experiences from unfinished features and provide clear context for the platform roadmap.
```
